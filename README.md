# 🚀 CareerConnect - Smart Job Portal

CareerConnect is a responsive Job Portal built using HTML, CSS, and JavaScript.

## Features

- User Login
- Job Listings
- Search Jobs
- Filter Jobs
- Save Jobs
- Apply for Jobs
- Career Journey
- Local Storage
- Responsive UI
- Dark Mode

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
